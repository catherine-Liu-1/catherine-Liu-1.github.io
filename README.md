# Catherine C. Liu Website

Static academic website for Prof. Catherine C. Liu.

## Pages

- `index.html` - biography, research interests, experience, education, and contact summary
- `research.html` - research areas, grants, professional activities, and opportunities
- `publications.html` - full publications list, book chapters, and manuscripts
- `people.html` - current students, postdocs, visiting students, and former students
- `news.html` - recent events and publication highlights
- `contact.html` - contact details and external profile links

## Deployment

This is a static site with no build step. It is intended to be deployed through GitHub Pages from the repository root.

For a user site, publish this repository as:

```text
catherine-Liu-1.github.io
```

The site will be available at:

```text
https://catherine-liu-1.github.io/
```
